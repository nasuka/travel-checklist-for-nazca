# Things to do
## Must
- [ ] 各種予防接種(黄熱、破傷風、A/B型肝炎、腸チフス)
- [ ] 各種通貨の両替(南米はよく停電が起きてATMが使えなくなるので重要。今回はペルー・ボリビアと、一応トランジットで寄るアメリカのものもあると安心)
- [ ] 各種持ち物の準備

## Want
- [ ] 海外旅行保険への加入(※1)



※1: 以下のサイトより、楽天カードを所持している場合は航空券を楽天カードで購入するか空港までの経路の料金を楽天カードで支払うことで海外旅行保険が付帯する。
